## About

This is the base image for the Laradock **Workspace** container — the all-in-one development shell that bundles PHP, Composer, Node, Git, and a large set of CLI tools, so you can run every command your project needs inside the container with nothing installed on your host.

Images are built and published automatically to [Docker Hub](https://hub.docker.com/r/laradock/workspace) for each supported PHP version.
